細菌試験ツール

【使い方】
1. index.html を開くと、パソコン内だけでも使用できます。
2. GitHub Pagesで公開する場合は、ZIPを展開して中の4ファイル
   index.html / style.css / script.js / data.js
   をリポジトリの一番上にアップロードしてください。
3. データを修正するときは data.js を編集します。

【GitHub Pages公開手順】
1. GitHubで新しいリポジトリを作成
2. Add file → Upload files
3. ZIPそのものではなく、ZIPを展開した中身をアップロード
4. Commit changes
5. Settings → Pages
6. Build and deployment の Source を Deploy from a branch
7. Branch を main、フォルダーを /(root) にして Save
8. 数分後に表示されたURLを開く

【注意】
画像から読み取った内容を初期データとして入れています。
業務で使用する前に、Excel原本と数値・表記を照合してください。
