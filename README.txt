細菌試験ツール v5

・「試験の種類（1）」で試験区分を選択します。
・ルーチンと定期試験ABは、選択後に試験の種類（2）ボタンを表示します。
・定期試験A・Bは、「試験の時期」を選択します。
・該当する試験種類が1種類だけの場合はボタンを省略し、そのまま結果を表示します。
  - 定期試験A：6月、9月、12月、3月
  - 定期試験B：6月、9月、12月、3月以外
・注記は「配水場出口水は含まれていません。」のみ表示します。

GitHubへはZIPを解凍し、中の index.html / style.css / script.js / data.js を上書きしてください。
